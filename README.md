# DockShield v5 — PierPressure Engine

React + Three.js + Supabase multiplayer marina simulator

## Features
- GLSL water shaders (fresnel, specular, foam)
- Procedural terrain
- Supabase Auth + Realtime multiplayer
- WebRTC spatial audio (PeerJS)
- B2B billboard monetization
- Persistent nautical miles
- Mobile touch controls

## Deploy
Deployed on Vercel: https://dockshield-v5.vercel.app
